Question 1: What is DOM? What is the purpose of DOM?

Answer: DOM stands for Document Object Model which is an XML and HTML programming API. DOM represents how a document is accessed and handled, as well as its logical structure.

Question 2: How will you select HTML elements using DOM? Write the name of the DOM methods.

Answer: A single element's ID is the quickest way to find it in the DOM. Besides ID, we can also select class name, tag name, and selector of an html elements. The name of DOM methods are as follows:
1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()

Question 3: What is event bubble?

Answer: Event Bubbling is a kind of DOM. The event bubbles up to the element's parent and grandparent elements in the DOM tree until it reaches the root element once an element receives an event.

Qustion 4: What is a callback function? Why will you use it?

Answer: A callback function is a function that receives an argument from another function and is subsequently called from inside the outer function to finish a task or activity. The purpose of this function is to run code in response to an event.
